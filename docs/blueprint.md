# **App Name**: Quizify AI

## Core Features:

- PDF Upload and Parsing: Allows users to upload PDF question papers, automatically extracting questions, options, and associated images, and offering the option to upload an answer key PDF for auto-mapping of correct answers.
- Exam Environment Simulation: Replicates a CBT-style (Computer Based Test) exam interface, incorporating elements such as a user profile photo, timer, on-screen calculator, navigation controls (Next, Previous, Save & Next, Bookmark Question), and a clear question status panel.
- Adaptive Testing: Implements an adaptive mode where the difficulty level of questions dynamically adjusts based on the user's ongoing performance, ensuring a personalized and challenging test experience.
- AI-Powered Answer Explanation: Generates step-by-step solutions using AI tool.
- Exam Mode Selection: Offers different exam modes like Practice Mode (instant feedback), and Exam Mode (strict testing), and Adaptive Mode (AI adjusted).
- Detailed Analytics Dashboard: Presents an auto-generated report post-submission, providing scores, accuracy rates, question-wise solutions, and progress graphs, helping users identify strengths and weaknesses.

## Style Guidelines:

- Primary color: Blue (#007BFF) for the main interactive elements to provide a sense of trust.
- Secondary color: Green (#28A745) to indicate success or correct actions, offering positive reinforcement.
- Tertiary color: Purple (#6F42C1) to provide a sense of creativity and innovation.
- Accent color: Orange (#FFC107) to highlight important CTAs and alerts, drawing user attention when necessary.
- The Home page will mimic the CBT Portal layout provided in the reference image, with a left sidebar for navigation, a main dashboard area showcasing key metrics (Available Exams, Completed Exams, Average Score, Questions Solved), recent exam activity, a performance trend chart, and available exams listed on the right.
- Font: 'Inter' (sans-serif) for both headlines and body text to maintain a clean, modern, and highly readable interface. 'Source Code Pro' (monospace) to show code snippets.
- Use a consistent set of icons throughout the interface. Ensure that the icons are minimalistic, and easily understood to aid in navigation.
- Ensure a distraction-free interface.
